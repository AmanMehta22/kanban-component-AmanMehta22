## Setup

```bash
# install dependencies
npm install

# run dev Next.js app (demo)
npm run dev

# run Storybook
npm run storybook
```

## Project structure
- `src/components/KanbanBoard` - main components and stories
- `src/hooks` - custom hooks
- `src/utils` - utility functions

## Features included
- Drag & drop (HTML5 drag API)
- Add / Edit / Delete tasks via modal
- Responsive layout for mobile / tablet / desktop
- Keyboard accessibility (basic: focus + keyboard activation)
- Storybook with required stories (Default, Empty, Large, Mobile, Interactive)
- Tailwind CSS design tokens extended
